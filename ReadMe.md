# Task 6: House Price Prediction

## Objective
The objective of this task is to predict house prices based on housing features.

## Dataset
California Housing dataset from sklearn.

## Model
Linear Regression

## Work Done
* Loaded dataset using sklearn
* Converted dataset into pandas DataFrame
* Selected features and target variable
* Applied Linear Regression
* Evaluated model using MAE and RMSE

## Results
* Model achieved reasonable prediction accuracy
* Predictions closely matched actual values
* Positive correlation observed

## Tools Used
* Python
* Pandas
* Scikit-learn
* Matplotlib
